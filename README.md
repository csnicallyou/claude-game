# Epochs of Humanity / Эпохи Человечества

[![CI](https://github.com/csnicallyou/claude-game/actions/workflows/ci.yml/badge.svg)](https://github.com/csnicallyou/claude-game/actions/workflows/ci.yml)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

> Гранд-стратегия с глубокой симуляцией. Начинается с верхнего палеолита (~45,000 лет до н.э., Левант). Эпохи добавляются модулями.

**Статус:** pre-alpha, в активной разработке. v0.1 в работе.
**Лицензия:** MIT
**Стек:** Godot 4.3+ с C# (.NET 8)

---

## Что это

Гибрид трёх жанров:

- **Crusader Kings** — играешь конкретного вождя племени, дин­ас­тия и наследование, личные события.
- **Europa Universalis** — стратегический слой: дипломатия, миграции, культурное и технологическое развитие.
- **Songs of Syx** — тактический слой: ~500 индивидуальных жителей на твоей стоянке, назначение работ, бой.

Всё это в **палеолитической эпохе** с уникальной механикой: над объективной симуляцией (климат, биология, экономика) лежит **слой восприятия** — персонажи интерпретируют события через анимистическую/шаманскую/тотемическую линзу палеолита. Засуха = «дух реки разгневан». Эта линза эволюционирует с эпохами вместе с человечеством.

**Что отличает от референсов:**

- Реальный палеолитический сеттинг (а не Civ-стиль «прыжок через эпохи»).
- Два играбельных вида: Homo sapiens и Homo neanderthalensis.
- Симуляция-восприятие как отдельный архитектурный слой.
- Архитектура «эпоха = модуль» — игра будет расти от палеолита до современности и далее.

- **.NET 8 SDK** ([скачать](https://dotnet.microsoft.com/download/dotnet/8.0))
- **Godot 4.3+ .NET edition** ([скачать](https://godotengine.org/download))
- **Git**

Опционально:
- JetBrains Rider или VS Code с C# Dev Kit для редактирования кода
- `dotnet format` для форматирования

## Запуск

```bash
git clone https://github.com/<your-username>/claude-game.git
cd claude-game
godot --editor project.godot       # открыть в редакторе
# или
godot project.godot                # запустить игру
```

## Сборка билдов

```bash
# через Godot Editor: Project → Export
# или CLI:
godot --headless --export-release "Windows Desktop" build/EpochsOfHumanity.exe
godot --headless --export-release "Linux/X11" build/EpochsOfHumanity.x86_64
godot --headless --export-release "macOS" build/EpochsOfHumanity.app
```

## Структура проекта

```
core/            # ядро (engine-agnostic C#, без using Godot)
sim/             # общая симуляция (без Godot)
render/          # рендер на Godot
ui/              # UI на Godot Control
eras/            # эпохи как модули (paleolithic, далее...)
data/            # JSON-конфиги (моддабельны)
localization/    # i18n (ru.csv, en.csv)
assets/          # графика, звук, палитры
scenes/          # Godot-сцены (.tscn)
tests/           # xUnit-тесты
docs/            # дизайн-документы и исследования
```

Подробнее — в [CLAUDE.md](./CLAUDE.md).

## Архитектура — четыре закона

1. **Детерминизм.** Те же входы + сид → тот же результат, всегда.
2. **Sim ↔ Render разделение.** Симуляция не знает про Godot.
3. **Эпоха = модуль.** Ядро не знает о существовании эпох.
4. **Dual-Reality.** Над объективной симуляцией — слой восприятия персонажей.

Подробное обоснование — в [CLAUDE.md §4](./CLAUDE.md).
